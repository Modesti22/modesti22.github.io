# modesti22.github.io
## Como editar esto
1. Descarga VSCode y Git (usa la configuración default)
2. Abre Git Bash y corre `git clone https://github.com/Modesti22/modesti22.github.io.git`
3. Actualiza `index.html` con datos y asegúrate de ubicar documentos y todo en los fólderes correctos (si le das a chatGPT o Claude el archivo, te pueden ayudar con los cambios si estás perdido)
4. Revisa los cambios abriendo `index.html` en un browser (Chrome, Edge, etc.)
5. En Git Bash, corre `cd "<INSERTA DONDE ESTA ESTE FOLDER>"`
6. Cuando estés contento, corre `git commit -am "<inserta una explicacion CORTA de los cambios>"`
 - Si recien installas git, no va a functionar. Solo corre los comandos que te dice para terminar de configurar git
7. Corre `git push` 
8. Revisa la página como en 30 min
